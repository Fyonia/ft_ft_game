## كيف تلعب 🎲

1. اكتب أسماء الفريقين
2. اختر ٦ فئات بالضبط
3. اختر سؤال من اللوحة
4. جاوب قبل يخلص الوقت
5. استخدم زر "إظهار الإجابة" للحكم
6. أعطِ النقاط للفريق الصحيح
7. خلّص اللوحة بالكامل للفوز 🏆

### نظام النقاط


* الإجابة الصحيحة = الفريق يأخذ النقاط ✅
* الإجابة الغلط = صفر ❌
* لا أحد عرف = الكل يخسر كرامته فقط 🤷
